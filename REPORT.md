## День 2 — Начало разработки backend

### Цель дня
Создание основы серверной части проекта и реализация первых модулей бизнес-логики.

### Изучил:
	•	принципы REST API
	•	основы Express.js
	•	модульную архитектуру backend-приложений
	•	работу HTTP-запросов

### Реализовал:
	•	инициализацию backend-проекта на Node.js
	•	настройку Express-сервера
	•	модульную структуру проекта
	•	тестовый API endpoint для проверки работоспособности сервера
	•	модуль расчёта срока действия абонемента

### Проблемы и решения:
	•	возникла ошибка доступа при использовании порта 5000 на macOS

### Итог дня:
Backend-приложение успешно запускается, реализованы первые бизнес-модули, соответствующие техническому заданию. Проект готов к подключению базы данных и дальнейшему развитию.
